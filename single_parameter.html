continuate.single_parameter¶

Single parameter numerical continuation

Options¶

tangentspace_dmu : float
Infinitesimal of parameter \(d\mu\) for calculating \(dx/d\mu\)
class continuate.single_parameter.TangentSpace(func, x, mu, tangentspace_dmu, **opt)[source]¶

Bases: object

Tangent space at \((x, \mu)\)

Parameters:

func : (numpy.array, float) -> numpy.array

\(F(x, \mu)\), func(x, mu) must have same dimension of x

Attributes

H (numpy.array) Krylov-projected matrix of Jacobian \(DF(x, \mu)\)
V (numpy.array) Basis yielded by Krylov subspace iteration, i.e. satisfies \(DF(x, \mu)V = VH\).
tangent_vector (numpy.array) normalized tangent vector \(d\xi = (dx, d\mu)\), where \(dx/d\mu = -DF(x, \mu)^{-1}F(x, \mu)\). The sign is chosen as \(d\mu > 0\).

Methods

Initial parameter of continuation, and satisfies \(F(x, \mu) = 0\)

step length of continuation. To decrease the parameter, you should set negative value.
